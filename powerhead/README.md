        [cite_start]<li><strong>Turn Off:</strong> Press and hold the main power button until indicators turn off. [cite: 240]</li>

    <h3 id="operation">3. Operation Guide</h3>
    <p><strong>Starting the Tool</strong></p>
    <ol>
        [cite_start]<li>Depress the lock-off lever. [cite: 255]</li>
        [cite_start]<li>While holding the lever, pull the switch trigger to start. [cite: 255]</li>
        [cite_start]<li>Release the trigger to stop. [cite: 256]</li>
    </ol>
    <img src="images/fig-trigger.png" alt="Trigger Operation" class="fig-img">

    <p><strong>Active Feedback sensing Technology (AFT)</strong></p>
    [cite_start]<p>The tool features AFT which stops the motor if it detects a sudden drop in rotation (e.g., a kickback). [cite: 228] [cite_start]The caution lamp will blink <strong>Green</strong>. [cite: 227] [cite_start]Release the trigger and clear the tool before restarting. [cite: 229, 230]</p>

    <p><strong>Reverse Button (Debris Removal)</strong></p>
    <ol>
        [cite_start]<li>Tap the reverse button while the tool is stopped. [cite: 263]</li>
        [cite_start]<li>Pull the trigger; the tool rotates in reverse and stops automatically. [cite: 264]</li>
        [cite_start]<li>Tap the reverse button again to return to normal rotation. [cite: 261]</li>
    </ol>
